# Applications

This folder groups the two Streamlit apps while keeping project data, models, and outputs at the project root.

- `channel_analysis/droplets_channel_app.py`: droplet segmentation and channel measurement app.
- `publication_compiler/compile_droplet_publication_app.py`: publication tables and figures compiler.

From the project root, keep using:

```bash
./lancer_app.command
apps/publication_compiler/lancer_publication_compiler.command
```

Portable installers are available at the project root:

- `install_dependencies_mac.command`
- `install_dependencies_windows_conda.bat`
